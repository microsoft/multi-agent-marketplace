## Quick Start

### Installation

```bash
# Step 1: clone repo
git clone https://github.com/microsoft/multi-agent-marketplace.git

# Step 2
uv sync
```

### Run simulation

Run your first simulation!

```bash
uv run packages/magentic-marketplace/experiments/run_experiment.py example_data/mexican_3_9
```
